# 42 Discovery Piscine — HTML, CSS & JavaScript Exercises

Exercises completed during the 42 Firenze Discovery Piscine.

This repository contains small web development exercises focused on learning the fundamentals of front-end development through practical tasks.

---

## Technologies Used

- HTML
- CSS
- JavaScript

---

## Topics Explored

- Basic HTML page structure
- Images and links
- HTML tables
- CSS styling and layout
- Responsive design concepts
- Parallax scrolling
- Basic JavaScript DOM interaction

---

## Repository Structure

Each folder corresponds to a Discovery Piscine cell.

cell1-0   → HTML basics exercises  
cell2-0   → CSS layout and responsive design  
cell2-1   → Additional layout and styling exercises  
cell2-2   → Parallax scrolling exercise  
cell3-0   → JavaScript DOM interaction exercises  
rush-html → Final mini website project
