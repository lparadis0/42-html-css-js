# 42 Discovery Piscine — Web

Exercises completed during the **42 Firenze Discovery Piscine**.

This repository contains small web development exercises built with:

- HTML
- CSS
- JavaScript

## Topics explored

- Basic HTML structure
- Tables
- Responsive layouts
- CSS styling
- Parallax scrolling
- Basic DOM interaction with JavaScript

## Structure

Each folder corresponds to a Discovery Piscine cell.

Example:

cell1-1 → HTML basics  
cell1-5 → HTML tables  
cell2-0 → Responsive layout  
cell2-2 → Parallax effect  
cell3-1 → JavaScript interaction
